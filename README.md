# ToDoList

Программа позволяет создать простой список задач, в который можно добавлять и удалять задачи.

Все добавленные задачи хранятся в файле note1.txt.

Программа ищет и создает этот и все остальные файлы на каталог выше исполняемого файла. 

Тестовый файл прилагается.

Идея такова, что каждая добавляемая задача автоматически записывается в файл, получая при этом 

порядковый номер.

По этому же номеру можно удалять задачи из файла и редактировать их.

Во время редактирования задача перепечатывается. Порядковый номер при этом не меняется.

При удалении задачи программа записывает обновленный файл temp.txt, а затем содержимое 

оригинального файл заменяется на содержимое файла temp.txt.

Если необходим оригинальный файл, то он сохраняется в backup.txt.
